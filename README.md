# RNPV

React Native Path Visualization

A small and simple example app with navigation, map and map marker

---

Install and run on android

npm install -g react-native-cli
cd RN-Path-Visualization
npm install
react-native run-android

---

Install and run on ios

npm install -g react-native-cli
cd RN-Path-Visualization
npm install
cd ios
pod install
cd ../
react-native run-ios
